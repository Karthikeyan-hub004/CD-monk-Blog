Frontend Interview Practice Project
Overview
A modern frontend application built to demonstrate real-world UI development skills, component architecture, API integration, and common frontend workflows. This project is designed to showcase practical knowledge of React, state management, and responsive UI development for internship and entry-level frontend roles.

Features
- Built with modern React (functional components & hooks)
- Responsive UI styling using Tailwind CSS or component-based styling
- API integration using mock REST services
- Forms, validations, and UI interactions
- Reusable and modular components
- Debugging and UI improvements
- Fast development setup using Vite

Tech Stack
- Frontend: React, JavaScript (ES6+)
- Styling: Tailwind CSS / Component-based UI
- State Management: Local State / Redux Toolkit
- Data Fetching: Fetch API / Axios / TanStack Query
- Build Tool: Vite
- Mock Backend: json-server
- Version Control: Git & GitHub

Prerequisites
- Node.js (LTS recommended)
- npm or yarn

Installation
Clone the repository and install dependencies:
npm install OR yarn install

Run in Development Mode
npm run dev OR yarn dev
Open the local URL shown in terminal (usually http://localhost:5173 or http://localhost:3000)

Build for Production
npm run build OR yarn build

Preview Production Build
npm run preview OR yarn preview

Mock API Setup (json-server)
Ensure db.json exists in project root
Run: npx json-server --watch db.json --port 3001

Endpoints:
- http://localhost:3001
- http://localhost:3001/blogs

Project Structure
src/ -> components, pages, services, styles
public/ -> static assets
package.json -> scripts and dependencies
db.json -> mock API database

Learning Outcomes
- Component-based architecture
- API handling and async operations
- Responsive UI design
- Debugging and performance improvements
- Clean and reusable code practices
